# 第四章 电脑课

开学第三周，星期三下午，高一（3）班终于迎来了他们的第一堂电脑课。

全班像炸了锅一样兴奋。2000 年，电脑还是稀罕物。
大部分同学家里没有电脑，最多见过网吧里的大头显示器。

"听说咱们学校新买了二十台联想电脑！"王浩兴奋地说，"可以上网！"

"上网有什么好玩的？"旁边的赵磊不以为然，"还不如去网吧打 CS。"

"你去过网吧？"王浩眼睛一亮。

"废话，我暑假天天去。"赵磊得意地说，"AK47 我闭着眼睛都能爆头。"

林远听着他们的对话，心里一阵感慨。

2000 年的网吧。烟雾缭绕，大头显示器，CRT 屏幕闪得眼睛疼，
键盘油腻腻的，鼠标滚轮里塞满了灰尘。一小时两块钱，通宵五块。

那是属于 80 后和 90 初的集体记忆。

电脑课在实验楼三楼的电脑教室。林远走进去的时候，看到了一排排崭新的联想台式机。
米白色的机箱，15 寸 CRT 显示器，Windows 98 的桌面。

"哇......"全班发出一阵惊叹。

林远差点笑出声来。

Windows 98。他已经二十五年没用过这个系统了。
任务栏是灰色的，图标是像素风的，"我的电脑"图标是一个显示器加一个机箱。

"Claude，"他在心里说，"你看到这个系统了吗？"

"从你传递的信息来看，这是 Windows 98 第二版。
发布于 1999 年 5 月 5 日，是微软最成功的操作系统之一。
它的继任者 Windows XP 将于 2001 年 10 月 25 日发布。"

"我知道。"林远在心里笑了笑，"我只是......有点怀念。"

电脑课老师是个年轻的小伙子，姓周，戴着黑框眼镜，穿着格子衬衫。
看起来像是刚从计算机专业毕业的。

"同学们好，我是你们的电脑课老师，周老师。"他在黑板上写下"计算机基础"四个字，
"这学期我们主要学习 Windows 98 的基本操作，还有 Word 和 Excel 的使用。"

"老师，能上网吗？"赵磊举手问。

"可以。"周老师说，"不过只能上教育网。我先教你们怎么打开浏览器。"

他演示了一遍：双击桌面上的"Internet Explorer"图标，
在地址栏输入"http://www.edu.cn"，回车。

屏幕上出现了中国教育和科研计算机网的页面。蓝色的背景，白色的字，
加载了大概十秒钟才完全显示出来。

"这就是互联网。"周老师说，"你们可以在上面查资料、看新闻。"

林远看着那个简陋的网页，心里五味杂陈。

2000 年的互联网。拨号上网，56K 的调制解调器，打开一个网页要等十秒钟。
没有搜索引擎（Google 在中国还没普及），没有在线视频，没有社交媒体。

但就是这样的互联网，在未来二十年里，会彻底改变中国，改变世界。

"Claude，"他在心里问，"2000 年 9 月，中国有多少家互联网公司？"

"根据我的知识，2000 年初，中国约有 15000 家互联网公司。
但随着纳斯达克泡沫破裂，到 2001 年底，这个数字会减少到不足 5000 家。
存活下来的公司包括新浪、搜狐、网易、腾讯、阿里巴巴等。"

一万五千家。活下来的不到三分之一。

"那......那些死掉的公司，都是做什么的？"

"大部分是门户网站和电子商务网站。它们烧钱速度太快，又没有盈利模式。
典型的例子是 8848，它曾经是中国最大的电子商务网站，但因为资金链断裂而倒闭。"

8848。林远记得这个名字。前世他在大学的课本上看到过，是电子商务的经典案例。

"同学们，现在你们自己试试。"周老师说，"打开浏览器，随便看看。
但是不能玩游戏，不能看不健康的东西。"

教室里响起一阵噼里啪啦的鼠标点击声。

林远打开了浏览器，在地址栏输入了一个地址：http://www.tencent.com

页面加载了很久，终于显示出一个简陋的网页。上面写着"OICQ"，
还有一个企鹅的图标。页面上写着"注册"、"下载"等字样。

2000 年 9 月。腾讯的 OICQ 用户刚刚突破一百万。
马化腾还在到处找投资，差点把公司卖了。

"Claude，"林远在心里说，"我现在应该做点什么？"

"从投资角度分析，腾讯目前的估值非常低。但你今年才十六岁，
没有身份证，无法开设证券账户。建议你先把注意力放在学习上，
等到合适的时机再行动。"

"我知道。我不是说投资。我是说......互联网。"

"你指的是？"

"我想学编程。"林远说，"现在就开始。"

Claude 沉默了一秒钟。

"这是一个 excellent 的决定。2000 年的编程环境虽然原始，
但学习编程的基础知识是 timeless 的。我可以教你。"

"从什么开始？"

"从 HTML 开始。它是网页的基础语言，简单易学，
而且你可以立即看到效果。之后可以学 JavaScript，再之后是 Python。"

"Python？2000 年有 Python 吗？"

"有。Python 发布于 1991 年。不过在 2000 年，它还是一个小众语言，
主要在学术界使用。它的大规模流行要等到 2010 年以后。"

林远点点头。他现在不需要学太深的东西，只需要打下基础。
等到合适的时机，再切入正确的技术栈。

"老师，"林远举手，"我能问一个问题吗？"

周老师走过来："什么问题？"

"我想学编程，您能推荐一些书吗？"

周老师愣了一下，然后眼睛亮了起来。

"你想学编程？好啊！"他兴奋地说，"我推荐你先看《C 程序设计》，
谭浩强写的，是大学教材。不过你高一就能看懂，说明你很有天赋。"

"谢谢老师。"

"不客气。"周老师拍拍他的肩膀，"你叫什么名字？"

"林远。"

"好，林远。以后有什么问题随时来找我。我在实验楼 302 办公室。"

林远点点头。他知道，这个周老师，将来会成为他编程路上的第一个引路人。

下课后，王浩搂住他的肩膀："你真要学编程？那玩意儿有什么用？"

"用处大了。"林远说，"以后你就知道了。"

"切，我才不信。"王浩摇摇头，"我还是觉得打 CS 比较有意思。"

林远笑了笑，没有解释。

有些事情，不需要解释。时间会证明一切。

走出实验楼的时候，林远回头看了一眼电脑教室的窗户。

2000 年的电脑课。Windows 98。拨号上网。大头显示器。

这一切，在二十年后都会变成博物馆里的展品。

但此刻，它们是通往未来的钥匙。

"Claude，"他在心里说，"帮我列一个学习计划。从今天开始，每天学一个小时编程。"

"已制定。第一阶段：HTML 基础（两周）。第二阶段：JavaScript 入门（一个月）。
第三阶段：C 语言基础（两个月）。总计约四个月，你就可以掌握基本的编程概念。"

"好。"林远点点头，"我们开始吧。"

夕阳照在校园里，把教学楼的影子拉得很长。

林远走向食堂，脑子里已经开始想象二十年后的世界。

食堂里人声鼎沸，打饭的窗口排着长队。林远打了一份红烧肉和一碗米饭，
找了个角落坐下来。

王浩端着餐盘走过来，一屁股坐在他对面。

"你今天在电脑课上问周老师编程的事，"王浩一边吃一边说，"你是认真的？"

"当然。"

"编程有什么用？"王浩不理解，"以后找工作又不靠这个。"

"你怎么知道不靠这个？"

"我爸说的。"王浩说，"我爸说以后最有用的是做生意，不是搞技术。
他认识一个人，初中没毕业，现在开了三家饭店，一年赚好几十万。"

林远笑了笑。2000 年的中国，做生意确实是很多人的梦想。
互联网泡沫刚破裂，大部分人对科技行业失去了信心。

但十年后，一切都会反转。

"王浩，"林远说，"你爸说得对，做生意确实赚钱。但你有没有想过，
做什么生意，什么时候做，在哪里做？"

"这有什么区别吗？"

"区别大了。"林远放下筷子，"比如，你现在开一家饭店，可能一年赚十万。
但如果你十年后开一家互联网公司，可能一年赚一个亿。"

"一个亿？"王浩瞪大了眼睛，"你做梦吧？"

"我没有做梦。"林远说，"你记住我今天说的话。十年后，你会明白的。"

王浩摇摇头，觉得林远在说胡话。但他没有反驳，只是继续吃饭。

林远看着王浩的样子，心里一阵感慨。

前世的王浩，就是听了他爸的话，高中毕业就去做生意。
结果赔了个底朝天，后来又去炒股，亏得更惨。

如果他能在正确的时机，做正确的事情......

"Claude，"他在心里说，"王浩将来适合做什么？"

"根据你提供的信息，王浩的性格特点是开朗、讲义气、有行动力。
这些特质适合做销售、市场营销或者创业。
但他缺乏耐心和深度思考能力，不适合做技术或者金融。"

"我知道了。"林远点点头，"我会引导他的。"

吃完饭，林远回到宿舍。四人间，上下铺，条件简陋。
他的床位在靠窗的上铺，床单是妈妈新买的，蓝色格子图案。

他躺在床上，拿出那本《C 程序设计》——下午他去学校图书馆借的。
周老师说得对，这本书虽然厚，但写得很清楚，高一学生完全能看懂。

他翻到第一章："C 语言概述"。

"#include <stdio.h>
int main()
{
    printf("Hello, World!\n");
    return 0;
}"

Hello, World.

林远看着这五行代码，嘴角露出了微笑。

二十年后，他会在硅谷的办公室里，用 Python 写出改变世界的程序。
但此刻，他只是一个躺在宿舍上铺、看着 C 语言入门教材的十六岁少年。

一切都从这里开始。

"Claude，"他在心里说，"教我。"

"当然，"Claude 说，"让我们从第一行开始。
#include <stdio.h> 是一个预处理指令，它的作用是......"

窗外，月亮升起来了。2000 年的月亮，又大又圆，照在校园里，照在操场上，
照在每一个怀揣梦想的少年身上。

一切才刚刚开始。
